Data Analytics

Power BI Dashboards

Convirtiendo planillas fragmentadas en dashboards en vivo, listos para tomar decisiones, para equipos de operaciones y liderazgo.

Rol

Data Analyst

Cronograma

2021 - En curso

Herramientas

Power BI, DAX, SQL

Estado

En produccion

Descripción

Un trabajo recurrente construyendo dashboards de Power BI para equipos que tomaban decisiones a partir de planillas estaticas actualizadas manualmente. Cada dashboard reemplazo horas de reporting manual por una vista en vivo y de autoservicio del negocio.

El alcance fue desde trackers de KPI para un solo departamento hasta dashboards ejecutivos que consolidaban ventas, operaciones y datos financieros en un solo modelo.

Problema

Los reportes vivian en archivos de Excel desconectados, actualizados de forma inconsistente e interpretados de manera distinta por cada equipo que los tocaba. El liderazgo tomaba decisiones con numeros que muchas veces tenian una semana de atraso, sin una definicion compartida de lo que realmente significaba cada metrica.

Solución

Disene un modelo de datos centralizado por area de negocio, con una unica definicion acordada para cada KPI. Construi reportes interactivos de Power BI sobre ese modelo, con drill-downs, filtros y actualizaciones automaticas que reemplazaron las exportaciones estaticas.

Proceso

  • Entreviste a los stakeholders para definir las preguntas reales que el dashboard necesitaba responder
  • Modele los datos (esquema estrella, relaciones, tablas calculadas) para performance y claridad
  • Escribi medidas DAX para KPIs, tendencias y comparaciones contra periodos anteriores
  • Disene el reporte para la audiencia: un clic a la respuesta, no diez
  • Automatice la actualizacion de datos via Power Query y gateways programados

Tecnologías

Power BIDAXSQLPower QueryExcel

Resultados

El tiempo de reporting bajo de dias a minutos para la mayoria de los equipos. Los dashboards se volvieron la referencia por defecto en las reuniones semanales, reemplazando las exportaciones ad-hoc de planillas y reduciendo las discusiones sobre cual numero es el correcto.

Aprendizajes

La parte mas dificil de un dashboard rara vez es el DAX: es lograr que los stakeholders se pongan de acuerdo sobre que significa una metrica antes de construir un solo visual. La disciplina de modelado de datos al principio ahorra mucho mas tiempo que optimizar visuales despues.
